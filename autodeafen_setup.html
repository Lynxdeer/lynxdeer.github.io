How to set up AutoDeafen

2 Minute Video Guide


Text Guide (for the impatient)


1

Creating a Discord Application

For the mod to work, we need to create an application to interface with our discord client.
Go to https://discord.com/developers/applications.
Create a New Application and enter a name. The name doesn't actually matter, so just input whatever.

2

Setting up OAuth2

Now, we need to get our application's credentials: our Client ID and Client Secret.
Navigate to the OAuth2 tab on the left of the screen.

Copy the Client ID by pressing the Clipboard Button next to it.
Tab back into Geometry Dash and click the button that says Paste Client ID. You should see a string of numbers appear above it (your client id).

Get your Client Secret by pressing the "Reset Client Secret" button. If you have 2FA enabled, you'll have to authenticate first. Press the clipboard button to copy your client secret.
Tab back into Geometry Dash and click the button that says Paste Client Secret. You should see a string of letters and numbers appear above it (your client secret).

Finally, you'll have to add a redirect. Press the button that says Add Redirect and type in the url http://localhost:8000.

Side Note: This Client Secret is not the same as your discord client token. It is unique to the application you just created, and gives control of the application to whatever possesses it. Although it probably doesn't matter if it's leaked anywhere since (to my knowledge) you couldn't really do much with it since there is no bot connected to it, it's probably unwise to share it.
3

Giving your application permissions

Now that your application is connected, it needs permission to do its job. Pressing the Next button in-game after setting your client id and secret will bring you to a webpage to authorize the application.
Click the Authorize button- and you're all done!

Troubleshooting

If you get a popup like this, try restarting your game. If that doesn't work, try re-setting up.
